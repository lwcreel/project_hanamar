# Project Hanamar

Project Hanamar is a project intended to help me develop skills with systems programming languages like C++ and Rust and experiment with game development.

## Installation

Ensure that you have Rust installed and properly configured. Clone this repository and run the following commands from the root directory:

```bash
cargo run
```
